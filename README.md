## Project Structure

```
  |-- base              # Lua source for the Ghost engine
  |-- ghost-extensions  # supporting libraries for the Ghost engine
  |-- love              # Love2d source
  |-- mobile            # iOS, Android, and React Native source for the Castle app
  |-- openssl           # openssl lib for Ghost engine
```
